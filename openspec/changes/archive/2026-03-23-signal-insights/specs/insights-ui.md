# Spec: Insights UI

## Capability

Display pre-computed coin briefs and enrichment data in a mobile-first LiveView page.

## Requirements

### Route & Access

- New route: `live "/signals/:id/insights", SignalLive.Insights, :show`
- Lives within the existing pro subscription `live_session`
- Feature flag guard: if `:signal_insights` is disabled, show a "coming soon" message
- "Insights" link added to signal list (both desktop table and mobile cards) — only visible when flag is enabled AND enrichment exists for that signal

### Page Layout (Mobile-First)

Top to bottom:

1. **Back navigation**: "Back to Signals" link
2. **Header**: Symbol, name, current price (live from signal), CoinScanX rank badge
3. **Freshness bar**:
   - "Last updated X minutes ago"
   - "Next update in approximately Y minutes"
   - If enriched_at > 1 hour ago: show stale warning style
   - Computes from `enriched_at` + known polling interval (30 min)
4. **TL;DR card**: Prominent, visually distinct. Contains the `tldr` text from brief.
5. **Volume analysis section**: Contains the `volume_analysis` text from brief.
6. **Full analysis section**: Contains the `full_analysis` text from brief.
7. **Quick facts grid**: Key metrics with static inline explanations (hardcoded in template).
8. **Disclaimer footer**: Contains the `disclaimer` text from brief. Muted styling.

### Quick Facts (Static Explanations)

Each fact shows the value + a one-line explanation hardcoded in the template:

| Fact | Explanation (EN) | Explanation (ES) |
|------|-------------------|-------------------|
| Market Cap | How much the entire coin supply is worth. Bigger = more established but less explosive growth potential. | Cuanto vale toda la oferta del token. Mayor = mas establecido pero con menor potencial de crecimiento explosivo. |
| 24h Volume | Total value traded in the last 24 hours. Higher = more real buying and selling activity. | Valor total transado en las ultimas 24 horas. Mayor = mas actividad real de compra y venta. |
| Vol / MCap | Volume as a percentage of market cap. Most coins sit at 3-10%. Above 30% = very active. | Volumen como porcentaje de la capitalizacion. La mayoria de monedas estan entre 3-10%. Mas de 30% = muy activo. |
| Supply | What percentage of total tokens are circulating. Lower = risk of future dilution. | Porcentaje de tokens en circulacion. Menor = riesgo de dilucion futura. |
| Exchanges | How many places you can buy/sell. More = better liquidity = easier to exit your position. | En cuantos exchanges puedes comprar/vender. Mas = mejor liquidez = mas facil salir de tu posicion. |
| From ATH | How far the current price is from its highest ever. | Que tan lejos esta el precio actual de su maximo historico. |
| Category | The type of project (meme, DeFi, AI, etc.) | El tipo de proyecto (meme, DeFi, IA, etc.) |

### Language Selection

- Determined by a user-level locale preference (if it exists) or defaults to English
- If no locale system exists yet, default to English with a simple toggle on the page ("EN / ES")
- Template conditionally renders `brief_en` or `brief_es` and the corresponding static explanations

### Real-Time Updates

- Subscribe to `"signal_enrichments:updated"` PubSub topic
- When enrichment is updated for this signal, reload enrichment data and re-render
- Also subscribe to `"price_updates"` for live price in the header

### Empty / Error States

| State | Display |
|-------|---------|
| No enrichment data yet | "Insights are being generated for this signal. Check back in a few minutes." |
| Enrichment exists but no brief | Show quick facts with raw values, "Analysis is being generated..." for brief sections |
| Feature flag disabled | "Signal Insights is coming soon." |
| Signal not found | Redirect to signals list with flash error |

## Constraints

- No JavaScript hooks needed — pure server-rendered LiveView
- No external CSS libraries — use existing Tailwind setup
- Template must not assume enrichment exists (always guard with conditionals)
- Quick fact explanations are hardcoded strings in the template, not generated by Claude
